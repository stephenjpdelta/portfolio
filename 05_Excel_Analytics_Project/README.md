## Excel Data Analysis Project



### Overview

This project demonstrates the use of Microsoft Excel for data cleaning, analysis, and reporting. The focus is on producing clear, structured outputs from raw tabular data using built-in Excel functionality.



---



### Key Tasks

- Cleaning and organising raw data
- Using formulas for calculations and validation
- Creating summary tables and metrics
- Applying filters and sorting to explore patterns
- Presenting results in a clear, readable format



---



### Tools & Techniques

- Pivot tables
- Advanced Filtering
- Conditional formatting
- Excel formulas (e.g. IF, VLOOKUP/XLOOKUP, COUNTIF, SUMIFS)
- Structured worksheets for repeatable analysis
- Dashboard Refresh Macro
- Slicers


---



### Purpose

The project highlights practical Excel skills commonly required in data administration and analyst roles, with an emphasis on accuracy, clarity, and usability.
